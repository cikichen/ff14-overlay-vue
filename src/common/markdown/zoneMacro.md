# 帮助

## 自述

本项目意在使玩家不再受困于FFXIV客户端本身的限制，提供一种自由的保存/使用 `用户宏`/`场景标记预设`的方式。

## 使用

直接在ACT悬浮窗中添加本项目的链接即可。

## 依赖

[鲶鱼精邮差](https://github.com/Natsukage/PostNamazu/releases)
